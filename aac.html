AAC for people with severe motor dysfunction

ALS, AAC, Prototyping, User Testing, Data Analysis

Summary

Date: January 2014 - July 2015
Stakeholders: University Hospital of Uberlândia (Brazil), University of Uberlândia (Brazil), Université de Lorraine (France)
Service: User research, software design, digital prototyping, data analysis
Team size: 4 members working directly
Duration: 1 year and 6 months
Tools used: .NET, WPF, C#, Excel, SPSS

My roles: (1) Prototype User Interfaces on WPF for different modules of an augmentative and alternative communication software; (2) Prototype input devices to control the system using accelerometers, eye-trackers, switches, electromyography; (3) Plan and execute user tests; (4) Data analysis based on user activity log.

Results: Developed a prototype that allowed patients with ALS, hospitalized at the university hospital, to communicate with family members, caregivers and doctors. The different sensors allowed an easy customization for the users to control the communication software.
