Challenge

Design a dashboard that allows researchers to identify in a glance if a specific user or provider presents a significant data loss

About the client

The primary goal of the Do CHANGE project is to develop a health ecosystem for integrated disease management for hypertensive and cardiac patients.

The project includes a clinical trial that checks efficiency of the lifestyle change advice provided by researchers. Cardiac patients receive lifestyle advice, while their wearable devices collect activity data over up to 10 months.

Problem description

My team's task was to develop a dashboard that shows trial participant's data and allow researchers to identify missing data in a glance. The information comes from different devices and multiple users. Researchers were interested to observe inter and intradaily data.
