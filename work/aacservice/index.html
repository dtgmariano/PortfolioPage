Challenge

Design a digital prototype that

facilitates food-related activities

for a specific group of seniors.

What is it about?

On this project, we worked for the European Consortium Reach 2020, in collaboration with the TU München and the TU Eindhoven. Reach 2020 aims to help elderly by focusing on stimulating healthy eating among seniors.

What did we do?

(1) Prototype User Interfaces on WPF for different modules of an augmentative and alternative communication software; (2) Prototype input devices to control the system using accelerometers, eye-trackers, switches, electromyography; (3) Plan and execute user tests; (4) Data analysis based on user activity log.

What did we achieved by the end of the project?

At the end of the case, we developed a prototype that allowed patients with ALS, hospitalized at the university hospital, to communicate with family members, caregivers and doctors. The different sensors allowed an easy customization for the users to control the communication software.
